# 概要

[Qiita API v2](https://qiita.com/api/v2/docs) の `https://qiita.com/api/v2/items` エンドポイントより最新の記事一覧を取得しレンダリングする簡単なアプリです。

# 使用API・ライブラリ

- Hooks
  - useState
  - useEffect
- axios
- Bootstrap
